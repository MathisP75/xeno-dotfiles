# xeno-dotfiles

My personal i3 wm setup

- Wm: i3
- Compositor: Picom pijulius
- Font: Iosevka Nerd Font
- Bar: Polybar
- Menus: Rofi
- Spotify client: Spotify-tui
- Terminal: Alacritty
- Icons: Customized Papirus icons and Zafiro icons (in the rofi launcher)
